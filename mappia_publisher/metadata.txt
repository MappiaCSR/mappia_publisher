about=It is part of my 4 years research. The laboratory CSR-UFMG use the mappia in many projects (csr.ufmg.br/csr/projeto/), the most recents are: project for the world bank about amazon evaluation (csr.ufmg.br/amazones), and the fire risk areas identification in Brazilian cerrado (csr.ufmg.br/fipcerrado). Use the map layout created in QGIS and share it online with everybody. You just need to create a free GITHUB account. Mappia is an online mapping platform accessible for everyone. Easy and free. Make your QGIS projects accessible to everybody. It automatically configure your map to show online. You can customize the generated maps online interactions into an user friendly interface, instructions on (mappia.earth). The service allows to use how many maps you want, it doesn't require any server. The geographic data is stored in your Github Account and you have the total control of it. Combine maps from multiple users sources and create an interactive platform online. For creating enhanced map interactions check our API at: http://mappia.earth/.
